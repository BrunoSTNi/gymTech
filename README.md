# 🏋️ GymTech

Sistema Inteligente de Gestão de Academias desenvolvido em PHP utilizando arquitetura MVC.

O GymTech permite o gerenciamento completo de alunos, planos, exercícios, equipamentos, treinos e possui um sistema de geração automática de treinos utilizando Inteligência Artificial baseada no perfil do aluno.

---

# 🚀 Funcionalidades

## 👨‍💼 Administrador

- Dashboard completo
- Cadastro de alunos
- Cadastro de planos
- Cadastro de exercícios
- Cadastro de equipamentos
- Gerenciamento de treinos
- Relatórios
- Geração automática de treinos

---

## 👨‍🎓 Área do Aluno

- Dashboard exclusivo
- Primeiro acesso com troca obrigatória de senha
- Visualização do perfil
- Plano contratado
- Treinos personalizados
- Visualização detalhada dos exercícios
- IA para geração automática do treino

---

## 🤖 Inteligência Artificial

A IA cria automaticamente:

- Treino A
- Treino B
- Treino C
- Treino D
- Treino E

considerando:

- Objetivo
- Dias de treino
- Nível de experiência
- Limitações físicas

# ⚙️ Instalação

Clone o projeto

```bash
git clone https://github.com/BrunoSTNi/GymTech.git
```

Copie para:

```text
xampp/htdocs/GymTech
```

Crie um banco chamado:

```text
gym_system
```

Importe:

```text
database/gym_system.sql
```

Configure a conexão com o banco.

Inicie:

- Apache
- MySQL

Acesse:

```text
http://localhost/GymTech/public
```
